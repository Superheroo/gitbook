每天进步一点点

jQuery

初尝jQuery

原生JS写法：var js_div = document.querySelector('.div')

JQ写法：var jq_div = $(".div")

原生JS点击事件写法： 

obj.addEventListener("click", function() {
                alert("我是JS事件")
})

JQ点击事件写法：

obj.on("click", function() {
                alert("我是JQ事件")

            })

JS原生获取行内属性：

getAttribute() 方法返回指定属性名的属性值。

obj.getAttribute(name)

JQ获取行内属性：

obj.attr("name")

name必需。需要获得属性值的属性名称

.attr()

attr() 方法设置或返回被选元素的属性值。

获取匹配的元素集合中的第一个元素的属性的值 或 设置每一个匹配元素的一个或多个属性。

obj.attr( attributeName ) //attributeName：属性名

obj.attr( attributeName, value ) // value:属性的值。

obj.attr( attributeName, function )  //function返回属性值的函数。

该函数可接收并使用选择器的 index 值和当前属性值。

val()

val() 方法返回或设置被选元素的值。

原生JS获取元素值

obj.addEventListener("input",function(){
    console.log(obj.value)
})

JQ获取元素值

obj.on("input", function() {
    console.log(obj.val())
})

addClass()

addClass() 方法向被选元素添加一个或多个类

原生JS添加类名:

obj.classList.add("类名")

JQ添加类名:

obj.addClass("类名")

removeClass()

removeClass() 方法从被选元素移除一个或多个类。

原生JS删除类名：

obj.classList.remove("类名")

JQ删除类名：

obj.removeClass("类名")

hasClass()

hasClass() 方法检查被选元素是否包含指定的 class。

obj.addClass("my_color")
obj.hasClass("my_color") //判断有无
alert(obj.hasClass("my_color"))

弹出：true

toggleClass()

toggleClass() 对设置或移除被选元素的一个或多个类进行切换。

如果没有类名则添加，有类名则移除

height()

obj.height() 方法返回或设置匹配元素的高度。

height() 获取的是样式里面写好的高度

innerHeight()

obj.innerHeight() 方法返回第一个匹配元素的内部高度。

获取的高度为：content+padding

outerHeight()

obj.outerHeight() 方法返回第一个匹配元素的外部高度。

获取的高度为：content+padding+border

offset()

offset(value) 方法返回或设置匹配元素相对于文档的偏移（位置）

value:规定以像素计的 top 和 left 坐标。

  • 1.返回值是定位+margin

  • 2.不管box-sizing设置的是什么，返回的始终是content到页面左上角的距离

  • 3.不管设置是四个方位里面的哪个，始终返回的是top和left

  • 4.当要设置值，只支持top值和left值，并且只能是数值类型的值，不能使其他数据类型

position()

position() 方法返回匹配元素相对于父元素的位置（偏移）。

obj.position():返回的是css样式里面的position值

scrollTop()

scrollTop() 方法返回或设置匹配元素的滚动条的垂直位置。

append()

obj.append()是能够在obj后面添加新元素

JS原生添加新元素：

obj.appendChild(new_li)

JQ写法：

obj.append(new_li)

appendTo()

将匹配的元素插入到目标元素的最后面

prepend()

a.prepend(b):a添加到b的最后

obj.prepend()是能够在obj前面添加新元素

obj.prepend(new_li)

prependTo()

b.prependTo(a): 将b添加到a的最后

after()

after() 方法在被选元素后插入指定的内容。

在匹配元素集合中的每个元素后面插入参数所指定的内容，作为其兄弟节点。

before()

before()在匹配元素的前面插入内容

insertAfter()

在目标元素后面插入集合中每个匹配的元素,插入的元素作为目标元素的兄弟元素。

insertBefore()

在目标元素前面插入集合中每个匹配的元素,插入的元素作为目标元素的兄弟元素。

fadeIn()

通过淡入的方式显示匹配元素。

.each( function(index, Element) )

遍历一个jQuery对象，为每个匹配元素执行一个函数。

<ul>
    <li>foo</li>
    <li>bar</li>
</ul>

$( "li" ).each(function( index ) {
  console.log( index + ": " + $(this).text() );
});

例子：遍历三个div并设置它们的color属性。

<!DOCTYPE html>
<html>
<head>
  <style>
  div { color:red; 
        text-align:center; 
        cursor:pointer;
        font-weight:bolder;
        width:300px; 
       }
  </style>
</head>
<body>
  <div>点击这里</div>
  <div>改变颜色</div>
<script src="../js/jquery-2.1.0.js" type="text/javascript" charset="utf-8"></script>
<script>
    $(document.body).click(function () {
      $( "div" ).each(function (i) {
        if ( this.style.color != "blue" ) {
          this.style.color = "blue";
        } else {
          this.style.color = "";
        }
      });
    });
</script>
</body>
</html>
