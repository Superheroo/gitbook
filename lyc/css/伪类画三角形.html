伪类画三角形

把一个容器的宽高设置为0，然后再进行画三角形

正常一个容器拥有宽高为下图所示：

当一个容器的宽高都为0时，如下图所示：

四块黑色部分分别为：1为：border-top 2为：border-right 3为 : border-bottom 4为：border-left

当我们想画出一个3中的正三角形时，这时就应该设置下边框为：border-bottom:100px，2和4分别设置为：border-right:50px solid transparent和border-left: 50px solid transparent;

代码实现如下：

  • 1.正三角形 (3)

<!DOCTYPE html>
<html>

    <head>
        <meta charset="UTF-8">
        <title></title>
        <style type="text/css">
            .triangle {
                width: 0;
                height: 0;
                border-bottom: 100px solid red;
                border-left: 50px solid transparent;
                border-right: 50px solid transparent;
            }
        </style>
    </head>
    <body>
        <div class="triangle"></div>
    </body>

</html>
