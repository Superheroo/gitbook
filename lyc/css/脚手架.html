每天进步一点点

1.到当前目录cmd命令，输入cnpm install --global vue-cli,全局安装vue-cli

2.创建一个基于webpack模板的新项目vue init webpack my-project,my-project为项目名，可以自行修改

3.创建时，会提问一些配置上的问题，

project name:项目名

project description:项目描述

author：作者

vue 配置

是否需要安装vue-router vue路由的插件(yes)

是否需要安装ESLint ESLint的插件(检查代码)，推荐不添加后面的都是yes

最后一项是创建文笔自动运行npm install,可以选择no，然后自动运行，最后等待下载完毕

run npm install 这一步no，cd my-project cnpm install cnpm run dev

然后等待下载完成

4.运行项目，npm run dev

在项目文件中，main.js 是我们的入口文件，主要作用是初始化vue实例并使用需要的插件

index.vue是我们的主组件，所有页面都是在App.vue下进行切换的。其实也可以理解为所有的路由也是index.vue的子组件。所以我将router标示为index.vue的子组件。
