每天进步一点点

gulp的使用

  • 1.新建项目，在项目文件中新建目录src，把所有的文件（css,html,img,js,scss）放到src文件夹中，对应的文件夹放入对应的文件
  • 2.将gulpfile.js和package.json放在项目根目录中
  • 3.上传时忽略node_modules文件夹时的操作步骤：

(1) 在项目文件夹中右键新建Git Bash Here，在弹出的窗口中输入：touch .gitignore,然后在文件夹中会生成gitignore文件

(2) 打开gitignore后输入：node_modules*

  • 4.建完项目结构后，在根目录下输入cmd后执行命令：cnpm i,输入执行后会生成node_modules文件夹

  • 5.然后再输入gulp dev命令。

    在gulpfile.js文件代码中，如下：

    var reload = browserSync.reload;
gulp.task('dev_browserSync', function() {
    browserSync.init({
        port: 3030,
        server: {
            baseDir: sourceDir,
            index: "./html/second.html"
        }
    });
});

    index: "./html/second.html"中的second.html是写代码时浏览器自动刷新的页面，可以在这个页面中使用超链接，链接到需要查看的页面
