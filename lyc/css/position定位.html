每天进步一点点

position定位

position 属性规定元素的定位类型。

position: fixed (固定定位)

position: absolute (绝对定位)

position: relative (相对定位)

1我们给父元素设置相对定位（relative）时，不会脱离文档流，也就是说会在地面占有面积 2.我们给要移动的元素设置绝对定位时（absolute）时，他会脱离文档流，在页面不占据任何位置 3.当我们给某个元素设置绝对定位时，一定要给他的参照物设置相对定位，否则他的参照物会是html
