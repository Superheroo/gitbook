每天进步一点点

transform,transtion,animation

transform:变形

 rotate：旋转 如：rotate:(180deg);旋转180度

 skew：扭曲  （用得较少。） 

 scale：缩放   1（正常样子，原型）  .9（是原型的0.9倍）  1.2（原型的1.2倍）

 translate：移动     translate:(50%,50%)X轴移动50%，Y轴移动50%。

transition：过渡

 transition-property:过渡效果的 CSS 属性的名称  如：transition-property: all;

transition-duration:过渡持续的时间

transition-timing-duration:变换速率    linear:匀速   ease：逐渐变慢    ease-in-out：加到减

transition-delay：动画开始执行的时间

鼠标经过图片变大代码：

img{

cursor:pointer    //鼠标经过图片时，鼠标变成手指

transition:all 0.6s;

}

:hover img{

transform:sclae(1.2);

}

animation：动画

animation-name: 规定需要绑定到选择器的 keyframe 名称

animation-duration:规定完成动画所花费的时间，以秒或毫秒计。

animation-timing-function:规定动画的速度曲线。

animation-delay:规定在动画开始之前的延迟。

animation-iteration-count:规定动画应该播放的次数。

animation-direction:规定是否应该轮流反向播放动画。

简写：animation:play .5s ease-in-out

-webkit- Safari 和 Chrome浏览器

-moz- 火狐浏览器

-ms- IE 浏览器

-o- Opera
